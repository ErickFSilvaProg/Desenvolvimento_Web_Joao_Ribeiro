Palavras-chave de cores

São cerca de 140 cores, com nomes em letras minúscula, que identificam cada cor.

RGB

Red, Green, Blue

Função que permite usar 3 parâmetros:
p { color: rgb(0, 0, 0); }

RGBA

Red, Green, Blue, Alpha

Função que permite o uso de um quarto parâmetro: Transparência/Opacidade:
p { color: rgba(0, 0, 0, 0); }
(Os valores do 'alpha' variam entre 0 e 1.)

HSL

Hue Saturation Lightness

Função que permite usar 3 parâmetros:
p { color: hls(0, 0%, 0%); }
(Tonalidade: 0° a 359°. Saturação: 0% a 100%. Luminosidade: 0% a 100%)
